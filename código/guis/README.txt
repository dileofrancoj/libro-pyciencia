Instrucciones para ejecutar los ejemplos del capítulo de GUIs

- ir al directorio de códigos del capítulo (necesario para que las instrucciones 
  de abajo funcionen ok y porque los programas usan recursos que necesitan estar
  en el mismo directorio)

- crear un entorno virtual y activarlo

    python3 -m venv env
    source env/bin/activate

- instalar las dependencias necesarias

    pip install -r requirements.txt

- listo! ya sólo ejecutar cualquiera de los ejemplos

    python 09.complete.py
